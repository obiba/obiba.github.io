Mica is a powerful software application used to create data web portals for large-scale epidemiological studies or multiple-study consortia. Mica2 is the successor of Mica.

Mica helps studies to provide scientifically robust data visibility and web presence without significant information technology effort. Mica provides a structured description of consortia, studies, annotated and searchable data dictionaries, and data access request management.

It is built upon a multitier architecture consisting of a RESTful application server and clients for instance:

  • The Mica Server Web Application from which one can edit information pertaining to indiviudal studies and datasets, and administer secure connections to the other applications of OBiBa software stack. {% include themes/bootstrap/image-item.html src='/assets/themes/bootstrap/img/mica-server-home.png' header='Mica Server Studies Page' caption='From the Mica Web Application, the user can administer the information partaining to studies, networks, datasets and data access requests.' %}
  • The Mica Drupal Client is used to create and display web data portals. It is built upon the Drupal content management system by means of Drupal modules. {% include themes/bootstrap/image-item.html src='/assets/themes/bootstrap/img/mica-client-home.png' header='Mica Portal Studies Page' caption='Thanks to the Mica Drupal Client, an entire web data portal can be easily customised to meet specific studies or consortia requirements.' %}
