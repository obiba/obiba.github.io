This is the RPM package repository of OBiBa software to automate installation and upgrade. To add this repository locally run the following command:

  wget https://bintray.com/obiba/rpm/rpm -O bintray-obiba-rpm.repo
  sudo mv bintray-obiba-rpm.repo /etc/yum.repos.d/

To verify that the repository has been added run:

yum list installed | grep bintray-obiba-rpm
On CentOS, install the package epel-release which will add repositories that provide some required packages, e.g. protobuf-python, R, etc.
On CentOS 6.x the following Python Clients dependencies must be installed manually:
  • Python 2.7
  • protobuf-python 2.5 or higher
  • python-pycurl
