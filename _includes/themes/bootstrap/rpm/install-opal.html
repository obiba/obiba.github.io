To install Opal: 

sudo yum install opal-server

To install Opal Python Client: 

sudo yum install opal-python-client
On CentOS 6.x the following Opal Python Client dependencies must be installed manually:
  • Python 2.7
  • protobuf-python 2.5 or higher
  • python-pycurl

To install R Server configured for Opal: 

sudo yum install opal-rserver
sudo service rserver restart

See Opal Documentation for more information.
