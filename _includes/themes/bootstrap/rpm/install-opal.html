To install Opal: 

sudo yum install opal-server

To install Opal Python Client: 

sudo yum install opal-python-client

To install R Server configured for Opal: 

sudo yum install opal-rserver
sudo service rserver restart

See Opal Documentation for more information.

To safely upgrade Opal <= 2.4.8 or rserver-admin <= 1.2.0, please remove the current installations as shown in the steps below before installing the new versions. The following releases address a packaging bug jeopardizing data folders upon removal: 

/etc/init.d/opal stop
rpm -e opal-server --noscripts
/etc/init.d/rserver stop
rpm -e rserver-admin --noscripts
