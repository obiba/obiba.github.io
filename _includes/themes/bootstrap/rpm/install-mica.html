To install Mica: 

sudo yum install mica2

To install Mica Python Client: 

sudo yum install mica-python-client
On CentOS 6.x the following Mica Python Client dependencies must be installed manually:
  • Python 2.7
  • protobuf-python 2.5 or higher
  • python-pycurl

See Mica Documentation for more information.
