The main new feature is the introduction of a new type of document: Research Project. A Research Project is the complementary of the Data Access Request. The Data Access Request is meant for setting up a collaboration between a researcher and a consortium/study whereas the Research Project allows to publish the results/status of this collaboration to the public. The Research Project document type is also highly extensible through a schemaform configuration.

Mica2 server upgrade from 1.2 will not affect the data. For the Mica Drupal modules and theme, see upgrade instructions .
