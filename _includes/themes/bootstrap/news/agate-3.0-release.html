This Agate release requires Java 21.

Upgrade actions - If the default Java installed in your system is not Java 21, you must install Java 21 (any vendor), update the path to the java executable in the file /etc/default/rock, and restart the agate service.

As an example: 

# location of java
JAVA=/usr/lib/jvm/java-21-openjdk-amd64/bin/java

Additional notes:

  • No upgrade action required when Agate is deployed with Docker, just pull the latest image.
  • The Agate API has NOT been modified: it will seamlessly integrate with Mica and Opal.

See also Agate documentation for installation and operation instructions.
