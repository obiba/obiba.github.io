OBiBa online sponsoring is now opened!

You are an happy OBiBa user willing to contribute? Several options are proposed:

  • Source code contribution. OBiBa is open source software then any contribution is welcomed: visit the OBiBa GitHub repository, fork, modify and make pull requests, we will be happy to review them! Before diving into major works, contact us to share your ideas so that we can guide you in a successful development.
  • Sponsor contribution. You can donate some financial resources so that we can maintain our test/demo server.
