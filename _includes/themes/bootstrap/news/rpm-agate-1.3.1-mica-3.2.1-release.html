Please note that Agate RPM packages prior to 1.3.1 and Mica RPM packages prior to 3.2.1 had an upgrade error breaking your subsequent installations. If you have an older version of Agate and Mica, you must run this script for Agate , this other script for Mica , before installing the new version. These scripts remove the older version safely and back up your config files under /usr/obiba_archives. Once you install the new version of Agate and Mica, you can retrieve all your config files manually.

Any updates made after installing Agate 1.3.1 and Mica 3.2.1 do not require this fix.

This fix applies only to RPM packages on CentOS/Fedora, they do not apply to Debian packages.
