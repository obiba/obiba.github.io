We are pleased to announce that Opal 5.3 is now available. Opal is OBiBa’s core data management application for biobanks.

This Opal 5.3 release requires Java 21. Please upgrade to 5.0 before upgrading to 5.3.

This release enhances support for Kubernetes environments. Creating a new R server session can be done asynchronously, allowing to follow up the different phases of the pod deployment (node readiness, pod scheduling, application start, etc.).

Opal also adds built-in support for MS SQL Server as a database backend. For those interested in using MS SQL Server as a "resource", see the R package odbc.resourcer (to be published soon in the CRAN).

This release was possible thanks to the funding of the DARE UK - Focus 5 program.
