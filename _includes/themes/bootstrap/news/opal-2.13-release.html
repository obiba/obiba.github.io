We are pleased to announce that Opal 2.13 is now available. Opal is OBiBaâ€™s core database application for biobanks.

Thanks to the support of the Canarie program, this release extends the datasource plugin service interface with plugins using the R server to import or export data:

  • from/to a delimited format file (ReadR based on readr ),
  • from/to a Excel file (ReadXL based on readxl and on writexl ),
  • from a Google Sheet (GoogleSheets based on googlesheets4 ).

A new analysis plugin service interface has also been added to integrate in the user interface data/meta-data analysis routines. The analyses are designed like a per-table test suite with a output report. As a proof of concept the plugin Validate based on the R package validate has been developed.

Plugins are available from our plugin repository . Please contact us if you want to develop and refer a new plugin. Using R packages to import/export and analyse data opens a lot of possibilities!
