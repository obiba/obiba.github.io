Rock is a R server that is accessible through a REST API.

Rock is the result of the refactoring of the "Opal's R server" administration application. It is now a fully independent application that can be used directly (see rockr R package) or as the computation backend of an application.

Opal 4.0 uses Rock for providing a scalable and reproducible computation environment for science.

This release was possible thanks to the funding of the EUCAN Connect program.
