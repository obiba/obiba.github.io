We have migrated our infrastructure from self-hosted servers to software-as-a-service (saas) open source software offers:

  • Source code, issue management and release packages are in GitHub ,
  • Continuous integration is performed by Travis-ci ,
  • Snapshot maven artifacts are in Artifactory repositories ,
  • Released maven artifacts and debian/rpm packages are in Bintray repositories ,
  • Documentations are hosted by Read the Docs ,

As a consequence system administrators must update Debian and/or RPM package repositories references: see the Debian repository and RPM repository instructions.

See also the Infrastructure description for more details.
