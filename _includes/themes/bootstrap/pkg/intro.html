This is the Debian package repository of OBiBa software to automate installation and upgrade. To use this repository, first add the key to your system: 

sudo apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 379CE192D401AB61

Then create a APT list file for OBiBa in /etc/apt/sources.list.d/obiba.list: 

echo 'deb https://dl.bintray.com/obiba/deb all main' | sudo tee /etc/apt/sources.list.d/obiba.list

Update your local package index: 

sudo apt-get update
Please note that the package apt-transport-https is required for apt to be able to handle repositories with a https URL.
