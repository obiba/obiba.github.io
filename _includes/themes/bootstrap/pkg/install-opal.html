To install Opal: 

sudo apt-get install opal

To install Opal Python Client: 

sudo apt-get install opal-python-client

To install Jennite VCF Store plugin: 

sudo apt-get install jennite-vcf-store
sudo service opal restart

To install R Server configured for Opal: 

sudo apt-get install opal-rserver
sudo service rserver restart

See Opal Documentation for more information.
