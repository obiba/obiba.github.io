Using Mica, studies can create out-of-the box a customizable web site including the following features:

Study Catalogue

Mica provides study networks with a study catalogue module to describe in a standardized manner their membership. Mica study catalogue includes:

  • A default list of important study description characteristics that can be customized to reflect specific requirements,
  • Support for longitudinal study design with data collection events, sub-populations and recruitment timelines.
  • Advanced study search functionality using facets technologies.

Datasets

Studies makes available different datasets to the research community. Mica provides:

  • A standard menu for its data resources,
  • A detailed description of a dataset including a comprehensive dictionary of all the variables provided by the dataset,
  • Advanced variable search functionality using facets technologies.

Importantly, Mica does not store any record-level data itself. Instead, Mica includes for studies using the OBiBa Opal software as main secured database solution, a distributed query engine that enables authorized researchers to obtain real-time aggregated reports on the exact number of participant with specific characteristics or phenotypes. The same engine is used to obtain summary statistics on each variable. Data Access Management

Mica provides a set of forms and workflows that allows a study to handle access requests electronically. Using the Mica Data Access module, single studies or consortia can:

  • Present their Data Access Policy,
  • Manage Data Access Requests from submission to approval,
  • Extend default Data Access Request form to match their needs,
  • Present the Data Access Committee members.

Research

Mica web portals offers sections to report on research activities undergoing within the study or the consortium. This includes pages to present the approved Data Access Requests and the scientific publications produced by the studies

Community

  • Present Contacts and Institutions that are part of the study or the consortium,
  • Use Blogs to allow studies to contributes to the life of the consortium,
  • Use Forums to offer a discussion space to your members,
  • Announce meetings and events.

Documents

  • Present Consortium documents with a full description and make them available for download.

Administration

  • Administer your Consortium web portal from the website itself,
  • Manage users and roles,
  • Define permissions on any kind of content,
  • Custom style and page layout.
