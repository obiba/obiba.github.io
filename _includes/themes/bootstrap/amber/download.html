Requirements

Amber Server is a stand-alone web server application, based on Nodejs.

Amber Studio, Amber Collect and Amber Visit are single page applications (javascript) that will be served by a web server such as Apache or Nginx.

To simplify the deployment and the customization, the recommended approach is to use Docker containers.

Instructions

Amber Server, Studio, Collect and Visit are available as Docker images, that can be overlayed.

For a quick start, we provide a Amber template project, which can be built and immediately tested. The build instructions can be found in the project's README file.
