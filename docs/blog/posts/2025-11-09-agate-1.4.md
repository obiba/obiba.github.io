# Agate 1.4 Released

We are pleased to announce that Agate 1.4 is now available. [Agate](/pages/products/agate/) is OBiBa’s central authentication server.

This is mainly a maintenance release: dependencies have been upgraded (more specifically the javascript UI framework).

New notification email templates have also been added to address [Mica](/pages/products/mica/)'s new feature (version 3.4.0, to be released): data access requests can be amended.

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
