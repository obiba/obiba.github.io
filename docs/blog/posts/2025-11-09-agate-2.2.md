# Agate 2.2 Released

We are pleased to announce that Agate 2.2 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

A new application setting was added to handle the users who sign-up through the application (like Mica): the user can either be automatically approved or be in the state of pending for approval (thus requiring manual approval by an administrator).

This release was possible thanks to the funding of the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) program.

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
