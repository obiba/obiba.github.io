# Mica2 1.0 Released

We are pleased to announce that Mica2 1.0 is now available. Mica is OBiBa’s data web portal for biobanks.

Mica2 is now officially available as a replacement for Mica. This new architecture is more robust and more domain-driven oriented. The web portal is still based on Drupal, but the Mica domain-specific part is now in an external application: Mica2 server. Mica2 takes also advantage of Opal for the data management (dictionnaries, summaries, access controls, taxonomies).

You can see Mica2 in action as the [The Canadian Partnership for Tomorrow Project ](https://portal.partnershipfortomorrow.ca) data portal and the [Maelstrom Research ](https://www.maelstrom-research.org/) web site.

See also [Mica documentation ](http://wiki.obiba.org/display/MICADOC) for installation and operation instructions.
