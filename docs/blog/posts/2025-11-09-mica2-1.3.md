# Mica2 1.3 Released

We are pleased to announce that Mica2 1.3 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for biobanks.

The main new feature is the introduction of a new type of document: Research Project. A Research Project is the complementary of the Data Access Request. The Data Access Request is meant for setting up a collaboration between a researcher and a consortium/study whereas the Research Project allows to publish the results/status of this collaboration to the public. The Research Project document type is also highly extensible through a [schemaform ](http://schemaform.io/) configuration.

Mica2 server upgrade from 1.2 will not affect the data. For the Mica Drupal modules and theme, see [upgrade instructions ](http://wiki.obiba.org/display/MICADOC/Mica+Drupal+Client+Installation+Guide#MicaDrupalClientInstallationGuide-DrupalModulesandThemeUpgrade).

See also [Mica documentation ](http://wiki.obiba.org/display/MICADOC) for installation and operation instructions.
