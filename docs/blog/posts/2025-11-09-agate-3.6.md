# Agate 3.6 Released

We are pleased to announce that Agate 3.6 is now available. [Agate](/pages/products/agate/) is OBiBa’s
  central user profile server.

Since Agate 3.0, this release requires **Java 21**. If you are still using Java 8, you must upgrade to Agate 3.0
  first.

New features and changes:

  - 
    The reCaptcha is now optional (default is enabled).

  - 
    Various maintenance updates.

See also Agate documentation  for installation and operation instructions.
