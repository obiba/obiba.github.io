# Mica2 4.4 Released

We are pleased to announce that Mica2 4.4 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces the following features in the public pages (pages of the published content):

- **Lists**, selected variables can be added in a cart or saved into a named list.
- **Search**, many [new settings ](http://micadoc.obiba.org/en/latest/admin/pub-configuration.html#search-page-settings) for configuring the search request, results extraction, sorting and display. Coverage filters added (full coverage, subdomains with variables).
- **Variable summary statistics** are improved (% per missing/non-missing values, per harmonized variables).
- **Data Access Request archiving** is possible when the project is completed (project was approved and end date is expired).
- **Data Access Request report notifications** are sent to the Data Access Officers as well (requires Agate 2.1.3).
- **Templates** are more responsive, many robustness improvements.

This release also introduces the following features in the administration pages:

- **Contact Us notifications**, feature and recipients are configurable.

See [Public pages configuration ](http://micadoc.obiba.org/en/latest/admin/pub-configuration.html) in the Mica documentation for the details on how to configure and customize your data portal front-end.

This release was possible thanks to the funding of [CanPath ](https://canpath.ca/) (data cataloging and data access requests) and the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) programs and the contribution of [Maelstrom Research ](https://www.maelstrom-research.org/).

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions. Note that a [Docker Image Installation ](http://micadoc.obiba.org/en/latest/admin/installation.html#docker-image-installation) section has been added.
