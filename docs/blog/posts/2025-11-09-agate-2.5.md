# Agate 2.5 Released

We are pleased to announce that Agate 2.5 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

In this release:

- **Strong password policy**: strong password is now required for new users (min/max length, upper/lower case, digit, special character).
- **Redirect**: requests with redirect parameter are verified.
- **Cookies**: have Secure, HttpOnly and SameSite flags.
- **CSRF**: [Cross Site Request Forgery](https://owasp.org/www-community/attacks/csrf) attempts are detected and blocked.
- **Content sanitization**: user provided content sanitization to protect from code injection.

This release was possible thanks to the funding of the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) program, and to the contribution of [CARTaGENE ](https://www.cartagene.qc.ca/).

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
