# Mica2 1.2 Released

We are pleased to announce that Mica2 1.2 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for biobanks.

The main new features are:

    - New search interface and API, very flexible and performant,

    - Addition of graphics on Mica content: geographical distribution, study design distribution etc.,

    - Improved attached files navigation and access controls.

  Mica2 server upgrade from 1.1 will not affect the data. For the Mica Drupal modules and theme, see [upgrade instructions ](http://wiki.obiba.org/display/MICADOC/Mica+Drupal+Client+Installation+Guide#MicaDrupalClientInstallationGuide-DrupalModulesandThemeUpgrade).

See also [Mica documentation ](http://wiki.obiba.org/display/MICADOC) for installation and operation instructions.
