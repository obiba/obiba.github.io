# Onyx 1.10 Released

We are pleased to announce that Onyx 1.10 is now available. Onyx is OBiBa’s data collection application for biobanks.

The main new features are:

    - Upgrade of java runtime to java8,

    - External datasources cache support, improving performance and onyx system resilience to remote datasource failure.

See full release note [here](http://wiki.obiba.org/display/ONYX/2015/12/04/Onyx+1.10+Released)
