# Mica2 6.2 Released

We are pleased to announce that Mica2 6.2 is now available.
  [Mica](/pages/products/mica/) is OBiBa's data web portal for epidemiological studies and harmonization initiatives.

**This release introduces Mica external editors.** They have the same privileges as standard Mica users but can also create their own documents (Network, Initiative, Study, etc.).

To enable this role, administrators must create the mica-external-editor group in Agate.

  This release was possible thanks to the development contributions of
  Maelstrom Research 
  .

  See
  Mica documentation 
  for installation and operation instructions.
