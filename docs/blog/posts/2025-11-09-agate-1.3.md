# Agate 1.3 Released

We are pleased to announce that Agate 1.3 is now available. [Agate](/pages/products/agate/) is OBiBa’s central authentication server.

This is mainly a maintenance release: dependencies have been upgraded (more specifically the embedded jetty web server), bugs have been fixed and general robustness has been improved.

New commands have been added to the [Agate Python client ](https://wiki.obiba.org/display/AGATEDOC/Agate+Python+Client+User+Guide):

- Add/delete user,
- Add/delete group,
- Add/delete application.

Agate server upgrade from 1.2 will not affect the data. However some configuration files have been modified. See [upgrade to 1.3.x instructions ](https://wiki.obiba.org/display/AGATEDOC/Agate+Configuration+Files#AgateConfigurationFiles-UpgradingtoAgate1.3.x).

See also [Agate documentation ](http://wiki.obiba.org/display/AGATEDOC) for installation and operation instructions.
