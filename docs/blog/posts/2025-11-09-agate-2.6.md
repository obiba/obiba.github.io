# Agate 2.6 Released

We are pleased to announce that Agate 2.6 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

In this release:

  - 
    **Two-factor authentication**: 2FA can be enabled for users managed by Agate.

  - 
    **Multi-domain OIDC setup**: when the same Agate server is accessed from different URLs, the OpenID Connect mechanism behaves consistently.

  This release was possible thanks to the funding of the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) program, and to the contribution of [LUMC ](https://www.lumc.nl/).

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
