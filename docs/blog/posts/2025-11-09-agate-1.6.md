# Agate 1.6 Released

We are pleased to announce that Agate 1.6 is now available. [Agate](/pages/products/agate/) is OBiBa’s central authentication server.

This release introduces new and updated Data Access Rquest templates related to changes in [Mica 3.8](/pages/news/release/mica2/3.8/).

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
