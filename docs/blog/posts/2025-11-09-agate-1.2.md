# Agate 1.2 Released

We are pleased to announce that Agate 1.2 is now available. [Agate](/pages/products/agate/) is OBiBa’s central authentication server.

This release offers the possibility to:

- select the language of the interface,
- edit and extend the translations of the interface, including the signup form,
- define the preferred language of a user.

Agate server upgrade from 1.1 will not affect the data.

See also [Agate documentation ](http://wiki.obiba.org/display/AGATEDOC) for installation and operation instructions.
