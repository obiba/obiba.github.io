# Opal 2.0 Released

We are pleased to announce that Opal 2.0 is now available. Opal is OBiBa’s core database application
    for biobanks.

Opal 2.0 is a major release that introduces many features and bug fixes. This release also offers a
    complete rework of the web user interface.

The main new features are:

    - the concept of project (previously known as
        datasource) has been added,

    - edition of dictionaries is made easier,

    - improved data exploration,

    - improved management of identifiers mappings,

    - improved user management,

    - better integration with R,

    - many other server improvements, including the
        support of MongoDB.
