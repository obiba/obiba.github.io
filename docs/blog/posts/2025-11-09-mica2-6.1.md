# Mica2 6.1 Released

We are pleased to announce that Mica2 6.1 is now available. [Mica](/pages/products/mica/) is OBiBa's data web portal for epidemiological studies and harmonization initiatives.

This release introduces the ability to configure a default language from among those enabled in Mica. For example, if English and French are available, you can set French as the default so that all users see Mica in French unless they explicitly choose another language. To do so, edit the MICA_HOME/conf/application-prod.yml file and add:

```
locale:
  default: fr
```

You must restart Mica for the change to take effect.

This release was possible thanks to the development contributions of Maelstrom Research.

See Mica documentation for installation and operation instructions.
