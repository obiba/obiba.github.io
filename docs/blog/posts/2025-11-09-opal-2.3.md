# Opal 2.3 Released

We are pleased to announce that Opal 2.3 is now available. Opal is OBiBa’s core database application
    for biobanks.

The main new features are:

    - R integration improvements (session monitoring, large datasets handling),

    - addition of taxonomies, which are a simple way to describe variable attributes,

    - support for markdown format in variable attributes (edition and preview),

    - variable ordering.

See full release note [here](http://wiki.obiba.org/display/CAG/2015/02/26/Opal+2.3+released)
