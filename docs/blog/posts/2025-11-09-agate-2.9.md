# Agate 2.9 Released

We are pleased to announce that Agate 2.9 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

This release offers the possibility to white and/or black list the email domains that are allowed / forbidden to sign up.

This release was possible thanks to the funding of the [MIREC Biobank ](https://www.mirec-canada.ca/).

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
