# Agate 1.5 Released

We are pleased to announce that Agate 1.5 is now available. [Agate](/pages/products/agate/) is OBiBa’s central authentication server.

Thanks to the support of the [Canarie ](https://www.canarie.ca/) program, this release introduces the ability for Agate to delegate authentication to third party services such as LDAP/Active Directory or a SQL database or [OpenID Connect ](https://openid.net/connect/) providers (for instance [Keycloak ](https://www.keycloak.org/) is a well established open source solution). Agate can also act itself as an OpenID Connect provider.

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
