# Mica2 5.4 Released

We are pleased to announce that Mica2 5.4 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and harmonization initiatives.

This release includes the following features:

- **2FA enforcement**, requires **Agate 2.10** or later.
- **Data Access forms download in Word format**, is available to administrators and data access officers, to facilitate data access request review with field entries commenting.
- **Data Access identifiers** can be incremental, instead of being generated randomly.

This release was possible thanks to the funding of the [Leiden University Medical Center - LUMC ](https://www.lumc.nl/), and of the [MIREC Biobank ](https://www.mirec-canada.ca/).

See Mica documentation  for installation and operation instructions.
