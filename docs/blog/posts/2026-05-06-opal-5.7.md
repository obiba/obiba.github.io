# Opal 5.7 Released

We are pleased to announce that Opal 5.7 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application
  for biobanks.

This release adds the following features:

- **File bundle command** is used to prepare file archive before download. It is now performed asynchronously, to support networks where connection is closed when no activity is detected. Opal R and Python packages have been updated to support this new asynchronous file bundle command.
- **OpenTelemetry log appender** library was added but is not part of the default log configuration.

This release was possible thanks to the support of the [Leiden University Medical Center - LUMC ](https://www.lumc.nl/).
