# Opal 5.1
Released

We are pleased to announce that Opal 5.1 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application
  for biobanks.

This Opal 5.1 release requires **Java 21**. Please upgrade to 5.0 before upgrading to 5.1.

This release follows a penetration test performed by an external security company.
  The vulnerabilities found have been fixed (mainly related to the files management).
  Various dependencies have been updated to the latest versions.

Note: R/DataSHIELD and Pyhton libraries are fully compatible with this new version.

This release was possible thanks to the funding of the
  Leiden University Medical Center - LUMC  and
  Maelstrom Research .
