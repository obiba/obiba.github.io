# Mica2 3.8 Released

We are pleased to announce that Mica2 3.8 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces several changes to Data Access Request:

- New notification when a request is created.
- Project timeline within a request.
- Ability to change a project start date.
- Being informed about annual and final report dates.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions.
