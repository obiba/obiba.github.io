# Opal 5.2 Released

We are pleased to announce that Opal 5.2 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application
  for biobanks.

This Opal 5.2 release requires **Java 21**. Please upgrade to 5.0 before upgrading to 5.2.

This release adds support for [Kubernetes ](https://kubernetes.io/) environments. 
  System administrators can now deploy Opal on Kubernetes clusters, improving resource management.
  Rock R/DataSHIELD servers are also supported and pods can be started on demand by Opal, for scalable data analysis environments.

This release was possible thanks to the funding of the
  Leiden University Medical Center - LUMC  and of the 
  DARE UK - Focus 5  program.
