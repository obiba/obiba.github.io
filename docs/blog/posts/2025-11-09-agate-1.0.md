# Agate 1.0 Released

We are pleased to announce that Agate 1.0 is now available. Agate is OBiBa’s central authentication server.

Agate is the user registry of Mica (server and Drupal client) and can be also used as an additional user registry of Opal.

User join requests can be submitted by users, directly from the Agate web interface or through the Mica-Drupal web portal. Agate also offers services of email notifications for the applications.

You can see Agate in action as the [The Canadian Partnership for Tomorrow Project ](https://portal.partnershipfortomorrow.ca) data portal and the [Maelstrom Research ](https://www.maelstrom-research.org/) web site.

See also [Agate documentation ](http://wiki.obiba.org/display/AGATEDOC) for installation and operation instructions.
