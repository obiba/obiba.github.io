# Opal 2.4 Released

We are pleased to announce that Opal 2.4 is now available. Opal is OBiBa’s core database application
    for biobanks.

The main new features are:

    - support of a new SQL data schema that allows to import/export data from/to plain SQL tables,

    - PostgreSQL support,

    - improved data import performance,

    - RPM packaging for CentOS-like systems.

See full release note [here](http://wiki.obiba.org/display/CAG/2015/09/24/Opal+2.4+Released)
