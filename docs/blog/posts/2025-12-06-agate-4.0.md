# Agate 4.0 Released

We are pleased to announce that Agate 4.0 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

This release modernizes the UI stack with AdminLTE 4 (RC4) built on Bootstrap 5, bringing improved performance and updated design patterns.

**Breaking Change:** If you have customized Agate's public pages, template revisions are required. Bootstrap 4 classes must be converted to their Bootstrap 5 equivalents. Key changes include updated implementations for popovers and modals. AdminLTE 3 layout structure has also changed significantly, with renamed classes and a revised page layout hierarchy. Please refer to the AdminLTE upgrade guide in the resources below for the full list of changes.

**Important:** While jQuery is still included in this release for backward compatibility, it will be removed in a future version. Please migrate your customizations to use Bootstrap 5's native JavaScript API rather than relying on jQuery.
