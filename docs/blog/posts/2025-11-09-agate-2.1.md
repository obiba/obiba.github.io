# Agate 2.1 Released

We are pleased to announce that Agate 2.1 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

Agate server can be run in a context path: the server URL can now be https://example.org/agate and not just in a subdomain such as https://agate.example.org.

New email template added for the "Contact Us" page of Mica 4.3.x. This is to be used when a user (not necessarily authenticated) wants to reach Mica's administrators or data access officers.

This release was possible thanks to the funding of [CanPath ](https://canpath.ca/) (data cataloging and data access requests) and the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) programs.

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
