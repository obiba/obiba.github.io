# Mica2 5.3 Released

We are pleased to announce that Mica2 5.3 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and harmonization initiatives.

This release includes the new feature related to harmonization protocols, namely, the retrieval of harmonization status statistics per study-specific datasets. Currently, only a handful of cohorts use this feature, here are some examples:

- [Maelstrom Research ](https://www.maelstrom-research.org/dataset/canpath-bl-hrfq-hp/)
- [The Canadian Partnership for Tomorrow's Health (CanPath) ](https://portal.canpath.ca/dataset/cptp_bl_hrfq)
- [COVID-19 Immunity Task Force (CITF) ](https://portal-dev.citf.mcgill.ca/dataset/citf-bl-demographics-hp)

See the percentages under each study-specific dataset in the harmonization potential table. A mouse over will provide the detailed counts of each harmonization status.

This is primarily a back-end feature in response to specific requests from cohorts that have a lot of harmonization information. Currently, this feature is not implemented in the default Mica public pages but may be included in future releases depending on cohort feedback.

This release was possible thanks to the development contributions of [Maelstrom Research](https://www.maelstrom-research.org) .

See Mica documentation  for installation and operation instructions.
