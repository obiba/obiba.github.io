# Agate 2.8 Released

We are pleased to announce that Agate 2.8 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

New email templates were added in this release for Mica 5.1+:

- **Mica Preliminary form**
- **Mica End-user Agreement form**

This release was possible thanks to the funding of the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) program, and to the contribution of [MIREC ](https://www.mirec-canada.ca/) and [Maelstrom Research ](https://www.maelstrom-research.org/).

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
