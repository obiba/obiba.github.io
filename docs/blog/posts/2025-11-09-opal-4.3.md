# Opal 4.3 Released

We are pleased to announce that Opal 4.3 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application for biobanks.

This is mainly a maintenance release, that enhances the security of the server:

- **Strong password policy**: strong password is now required for new users (min/max length, upper/lower case, digit, special character).
- **Cookies**: have Secure, HttpOnly and SameSite flags.
- **CSRF**: [Cross Site Request Forgery](https://owasp.org/www-community/attacks/csrf) attempts are detected and blocked.
- **Plain R service**: a setting allows to permanently block the plain R service. The DataSHIELD service is still available and is the recommended one to operate on the R server side.

This release was possible thanks to the funding of the [EUCAN-Connect ](https://www.eucanconnect.eu/) program.
