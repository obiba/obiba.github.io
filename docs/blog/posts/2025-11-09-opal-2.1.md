# Opal 2.1 Released

We are pleased to announce that Opal 2.1 is now available. Opal is OBiBa’s core database application
    for biobanks.

The main new features are:

    - every type of data has a summary (including open text and geolocalisations),

    - DataSHIELD calls are parallelized: instead of waiting for each Opal to answer to the DataSHIELD request, (1) request is first submitted to each Opal and (2) results are fetched afterwards.
