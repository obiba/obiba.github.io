# Mica2 3.5 Released

We are pleased to announce that Mica2 3.5 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces the following:

    - 
      **Variable Lists**, enable the researcher to name and save search results for data exploitation which include: exploring data dictionaries, obtaining taxonomy coverage statistics, searching entities and binding variables to a data access request.

    - 
      **Variable Cart**, is an *unnamed* list that is bound to the current user and the browser. By default an anonymous Mica Portal user will have a variable cart.

Make sure to review and change the Lists and Cart configurations to adapt them to your needs.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions.
