# Mica2 3.7 Released

We are pleased to announce that Mica2 3.7 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces the ability to authenticate users via Agate's OpenID Connect Realms.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions.
