# Mica2 1.4 Released

We are pleased to announce that Mica2 1.4 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for biobanks.

Mica offers a better multi-language support both for the administration interface and for the published content. The user preferred language (see also [Agate 1.2](/pages/news/release/agate/1.2/) announcement) is handled accordingly.

The Data Access Forms specifications can now integrate more complex validations (word count, date comparison etc.) and can be printed directly (without the need of defining a PDF template).

Mica2 server upgrade from 1.3 will not affect the data. For the Mica Drupal modules and theme, see [upgrade instructions ](http://wiki.obiba.org/display/MICADOC/Mica+Drupal+Client+Installation+Guide#MicaDrupalClientInstallationGuide-DrupalModulesandThemeUpgrade).

See also [Mica documentation ](http://wiki.obiba.org/display/MICADOC) for installation and operation instructions.
