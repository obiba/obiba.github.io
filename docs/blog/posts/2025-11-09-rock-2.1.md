# Rock 2.1 Released

We are pleased to announce that Rock 2.1 is now available. [Rock](/pages/products/rock/) is OBiBa’s R computation engine.

This Rock release improves the performance (faster authentication operation).

Additional notes:

    - 
      No upgrade action required when Rock is deployed with Docker, just pull the latest image.

    - 
      The Rock API has NOT been modified: it will seamlessly integrate with **Opal**.

See also [Rock documentation ](http://rockdoc.obiba.org/) for installation and operation instructions.
