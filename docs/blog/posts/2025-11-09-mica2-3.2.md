# Mica2 3.2 Released

We are pleased to announce that Mica2 3.2 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces a new search user interface for the metadata catalogue with several user experience improvements:

- An improved classification browser for document properties as well as for classifications developed by [Maelstrom Research](maelstrom-research.org) such as Areas of Information.
- Better selection and filtering of classifications to specify search criteria.
- More intuitive interface to search for existing variables by name or label as specified in source questionnaires.
- Improved search performance and document listing.

Please note that installing or upgrading to Mica2 3.2 requires the latest version of the Mica search ES plugin 1.1.0
        
- [Mica search ES 1.1.0 (.deb)](http://pkg.obiba.org/stable/mica-search-es_1.1.0-b20180122120457_all.deb)
- [Mica search ES 1.1.0 (.rpm)](https://rpm.obiba.org/stable/mica-search-es-1.1.0-1.noarch.rpm)
- [Mica search ES 1.1.0 (.zip)](https://download.obiba.org/plugins/stable/mica-search-es-1.1.0-dist.zip)

Also see [Mica documentation ](http://wiki.obiba.org/display/MICADOC) for installation and operation instructions.
