# Mica2 4.1 Released

We are pleased to announce that Mica2 4.1 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces the following new features:

- **Study Import**, enables researchers to share study descriptions from remote Mica servers having identical data model - courtesy of [RECAP Preterm ](https://recap-preterm.eu/) and [EUCAN-Connect ](https://www.eucanconnect.eu/).
- **Persons Management**, administrators can now search, delete and update persons (contacts, investigators) associated with different studies and networks.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions. Note that a [Docker Image Installation ](http://micadoc.obiba.org/en/latest/admin/installation.html#docker-image-installation) section has been added.
