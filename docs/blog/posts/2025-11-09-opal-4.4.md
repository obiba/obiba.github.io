# Opal 4.4 Released

We are pleased to announce that Opal 4.4 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application for biobanks.

This release introduces the following new features:

- **Two-factor authentication (2FA)**: when the user activates 2FA for its account, an additional temporary key will be requested along with its username/password credentials (see [documentation](https://opaldoc.obiba.org/en/latest/web-user-guide/my-profile.html#two-factor-authentication)).
- **Resource views**: when a [resource](https://opaldoc.obiba.org/en/latest/resources.html) can coerce to a tabular representation, it is now possible to create a view over this resource in order to make the data and their dictionary more accessible for documentation (using Mica) or data extraction (see [documentation](https://opaldoc.obiba.org/en/latest/web-user-guide/projects/resources.html)).
- **Personal Access Token (PAT) expiration**: for extra security, an end of life can be defined for the PATs (default is to never expire) and an inactivity timeout is now applied (default is 2 months, with the possibility to renew it, see [PAT documentation](https://opaldoc.obiba.org/en/latest/web-user-guide/my-profile.html#personal-access-tokens) and [Login policy configuration](https://opaldoc.obiba.org/en/latest/admin/configuration.html#login-policy-configuration)).
- **Tasks execution per project**: tasks (import, export etc.) will be executed in parallel when they belong to different projects; tasks within a single project are still executed sequentially.
- **Project Backup/Restore**: the backup/restore functionality (tables, views, resource references, files, report templates) was only available through the Python and R interfaces. It is now possible to trigger these tasks from each project's administration page.

Opal Python command tool and [Opal R package](https://www.obiba.org/opalr/) have been updated according to these new features.

This release is expected to be fully backward compatible.
