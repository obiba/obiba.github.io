# Mica2 3.4 Released

We are pleased to announce that Mica2 3.4 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces the following new functionalities:

    - 

      **Data Access Request Amendments**, researchers can now request changes (e.g. additional data, new research collaborators) to an approved data access request. The data access amendment workflow is identical to that of data access requests and its web form can be customized by the Mica administrator.

    - 
      **Private Comments**, enhances collaboration between data access officers and committee members when processing a data access request or amendment. These comments are not visible to researchers.

    - 
      **Logging Actions**, data access officers and committee members can log actions taken on a data access request or amendment. These actions are saved as part of the data access request history.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions.
