# Agate 1.1 Released

We are pleased to announce that Agate 1.1 is now available. Agate is OBiBa’s central authentication server.

This release offers the possibility (for applications that are not part of the OBiBa software suite) to use Agate as a standard Authentication and ID provider.

The main new features are:

    - support of main [OAuth2](http://tools.ietf.org/html/rfc6749) specifications,

    - support of main [Open ID Connect](http://openid.net/specs/openid-connect-core-1_0.html) specifications,

    - Single Sign-On is available within OBiBa software suite.

See also [Agate documentation ](http://wiki.obiba.org/display/AGATEDOC) for installation and operation instructions.
