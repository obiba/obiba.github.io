# Mica2 4.0 Released

We are pleased to announce that Mica2 4.0 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

Public Pages

This release introduces a long-time waited feature: published content is now directly served by Mica, i.e. **Drupal is no more a requirement**.
  Mica now proposes a complete out-of-the-box solution for building and publishing data catalogs. The theme is based on the excellent [AdminLTE ](https://adminlte.io/) framework which itself uses [Bootstrap ](https://getbootstrap.com/) toolkit. This gives a modern and clean style to default Mica public pages, see [Mica demo ](https://mica-demo.obiba.org).

The public pages are based on templates that can be:

- **Configured**, by modifying the global Mica configuration, or by altering settings variables in a template file,
- **Extended**, by including new content in the base template files or by defining new pages,
- **Overridden**, by overridding the base template files.
- **Styled**, by altering the stylesheets or by defining a custom build of the AdminLTE distribution.

Data Access Request

This release introduces a new feature: feasibility inquiry forms. The purpose of this form is to request for a short-track evaluation of the feasibility of a research project. This evaluation will inform whether the data access request worths being submitted and can also be used as a letter of support from the data owner when submitting a research proposal to funding agencies.

**Important notes**

- Some of the features of the Drupal's public pages have not been implemented yet: attached files browser, variable cart, crosstabs, graphics and advanced search. Please contact us, to help with defining priorities.
- Note also that **this version of Mica does not exclude Drupal front-end from being used**, facilitating the transition to Mica's public pages.
- This release integrates better with [Agate 2.0](/pages/news/release/agate/2.0/)

This release was possible thanks to the funding of [CARTaGENE ](https://www.cartagene.qc.ca/) (data access requests), the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) and the [SYNCHROS ](https://synchros.eu) (cataloging) programs.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions. Note that a [Docker Image Installation ](http://micadoc.obiba.org/en/latest/admin/installation.html#docker-image-installation) section has been added.
