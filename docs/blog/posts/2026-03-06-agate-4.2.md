# Agate 4.2 Released

We are pleased to announce that Agate 4.2 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

This release is mainly focused on maintenance and security improvements.

This release was possible thanks to the development contributions of [Maelstrom Research ](https://www.maelstrom-research.org/).

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
