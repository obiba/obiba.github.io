# Opal 5.6 Released

We are pleased to announce that Opal 5.6 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application
  for biobanks.

This release is mainly focused on maintenance and security improvements.

Upgrade notes: when using Docker based deployment, make sure to upgrade to version 5.3.3 before migrating to 5.6.0 or higher.

This release was possible thanks to the funding of the
  DARE UK - Focus 5  program.
