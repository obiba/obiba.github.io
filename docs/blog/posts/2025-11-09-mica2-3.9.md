# Mica2 3.9 Released

We are pleased to announce that Mica2 3.9 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release enables the search engine to use the upcoming search plugin compatible with Elasticsearch 7.

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions.
