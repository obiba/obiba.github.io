# Mica2 4.2 Released

We are pleased to announce that Mica2 4.2 is now available. [Mica](/pages/products/mica/) is OBiBa’s data web portal for epidemiological studies and networks.

This release introduces the following new administration features:

  - 
    **Revision Comparison**, permits the comparison of two revisions of a Mica content document (Network, Study, etc).

  - 
    **Content Statistics Summary**, provides the count of all Mica content documents by revision status (Published, Under Review, In Edition, etc) as well as the count of all their associated files and variables.

  - 
    **Filtering by Revision Status**, Mica content document listing can be filtered by revision status..

See [Mica documentation ](http://micadoc.obiba.org) for installation and operation instructions. Note that a [Docker Image Installation ](http://micadoc.obiba.org/en/latest/admin/installation.html#docker-image-installation) section has been added.
