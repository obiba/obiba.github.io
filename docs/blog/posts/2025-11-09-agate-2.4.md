# Agate 2.4 Released

We are pleased to announce that Agate 2.4 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

In this release:

-  [OpenID Connect realms ](https://agatedoc.obiba.org/en/latest/web-user-guide/realms.html#open-id-connect-realm) username and groups extraction process can be configured,
-  Username can be an email,
-  Improved client IP detection.

This release was possible thanks to the funding of the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) program.

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
