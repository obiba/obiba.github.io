# Opal 5.5 Released

We are pleased to announce that Opal 5.5 is now available. [Opal](https://www.obiba.org/pages/products/opal/) is OBiBa’s core data management application
  for biobanks.

This release introduces security improvements, to conform best practices:

- **API documentation** is now available for better developer experience. All the REST endpoints are documented.
  The Swagger UI can be accessed from the Opal main Administration page.
- **User info tracking** has been added to log user information object when authenticated via OpenID Connect.
  The user profiles can be downloaded from the Opal main Administration page for further processing.
- **R/DataSHIELD authentication strategy** can be restricted to Personal Access Tokens (PAT) only, disallowing 
  username/password authentication. This enhances security by using tokens that can be individually managed and revoked 
  without affecting user credentials.

Upgrade notes: when using Docker based deployment, make sure to upgrade to version 5.3.3 before migrating to 5.5.0 or higher.
