# Opal 2.5 Released

We are pleased to announce that Opal 2.5 is now available. [Opal](http://www.obiba.org/pages/products/opal/) is OBiBa’s core database application
    for biobanks.

The main new features are:

    - improved variable taxonomies management, including integrated access to the official [Maelstrom Research variable taxonomies](https://github.com/maelstrom-research/maelstrom-taxonomies),

    - (very) large derivation scripts are now supported,

    - improved performance of tabular SQL databases (millions of entities can be handled).

See [Opal documentation](http://wiki.obiba.org/display/OPALDOC) for installation and operation instructions.
