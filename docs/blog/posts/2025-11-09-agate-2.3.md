# Agate 2.3 Released

We are pleased to announce that Agate 2.3 is now available. [Agate](/pages/products/agate/) is OBiBa’s central user profile server.

In this release:

- Best efforts are done to get clients' IP, despite the reverse proxies, so that it appears in the web services resquests log (rest.log),
- Dependency on Google fonts service removed, so that pages work in restricted environments.

This release was possible thanks to the funding of the [EUCAN Connect ](https://www.eucanconnect.eu/) ([Coral](https://www.eucanconnect.eu/coral-docker-stack-deployment/) integration) program.

See also [Agate documentation ](http://agatedoc.obiba.org/) for installation and operation instructions.
